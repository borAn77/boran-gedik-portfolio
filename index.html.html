Who I Am

I'm Boran Gedik, a Computer Science student based in Wrocław, Poland, studying at Coventry University Wrocław.

I started programming because I wanted to build real products and understand how technology works behind the scenes. Since then I've been focused on learning by building — projects over pure theory.

Over the past year I've developed more than 15 programming projects while improving my skills in backend development and automation systems.

My main programming language is Python, but I also work with JavaScript and web technologies. I'm particularly interested in backend architecture, automation workflows, and AI-powered tools.

Some of the projects I've built include lead generation tools, automation systems, and full websites for real users. My long-term goal is to become a highly skilled software engineer and build scalable technology products.
